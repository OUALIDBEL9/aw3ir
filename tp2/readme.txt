TP2 - Projet Web (HTML / CSS / JS)
Fichiers fournis:
- index.html : formulaire vif et responsive
- style.css  : styles, validation visuelle
- action.html : page qui lit l'URL et affiche les données
- action.js  : script de lecture d'URL
Instructions:
1) Ouvrir index.html dans un navigateur (double-cliquer suffit).
2) Remplir le formulaire (tous les champs requis) et cliquer sur Envoyer.
3) La page action.html affichera les valeurs envoyées via GET.
